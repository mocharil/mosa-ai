## ğŸ’¡ Tips Membaca Script

### âœ… DO:
1. **Bicara natural** - Jangan kaku, seolah ngobrol biasa
2. **Jeda setelah pertanyaan** - Beri waktu audience lihat response
3. **Highlight angka** - "Cuma 1-2 detik" lebih impactful dari "cepat"
4. **Senyum** - Voice demo lebih engaging kalau presenter enjoy
5. **Repeat key points** - Fast, empathetic, intelligent

### âŒ DON'T:
1. Jangan baca terlalu cepat
2. Jangan skip waiting time - biar audience lihat prosesnya
3. Jangan lupa aktifkan demo mode
4. Jangan monoton - variasi intonasi
